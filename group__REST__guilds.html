Requires Permission::BanMembers permissions. You can additionally specify deleteMessageDays to remove user messages for last (0-7) days.

◆ createGuild()

nlohmann::json Hexicord::RestClient::createGuild ( const nlohmann::json &  newGuildObject)

Create a new guild.

Returns a guild object.

See https://discordapp.com/developers/docs/resources/guild#create-guild-json-params for list of fields to pass.

Warning
By default this endpoint is limited to 10 active guilds. These limits are raised for whitelisted GameBridge applications.
Note
If roles are specified, the required id field within each role object is an integer placeholder, and will be replaced by the API upon consumption. Its purpose is to allow you to overwrite a role's permissions in a channel when also passing in channels with the channels array.

Throws RESTError if invalid IDs specified or current user don't have required permissions. Also may throw boost::system::system_error if client fails to connect to REST API server.
