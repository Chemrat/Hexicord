Send raw REST-request and return result json.

Note
Newly constructed RestClient object don't have open REST connection. It will be openned when restRequest called first time.
Parameters
methodused HTTP method, can be any string without spaces but following used by Discord API: "POST", "GET", "PATCH", "PUT", "DELETE".
endpointendpoint URL relative to base URL (including leading slash).
payloadJSON payload, pass empty object (default) if none.
queryGET request query.
multipartPass one or more MultipartEntity to perform multipart request. If payload is also present, it will be first entity.
Exceptions
RESTErroron API error.
boost::system::system_erroron connection problem.
