Delete a message.

If operating on a guild channel and trying to delete a message that was not sent by the current user, requires the 'MANAGE_MESSAGES' permission.

Fires MessageDelete event.

Exceptions

void Hexicord::RestClient::deleteMessages ( Snowflake  channelId,
const std::vector< Snowflake > &  messageIds 
)

Delete multiple messages in a single request.

This endpoint can only be used on guild channels and requires the 'MANAGE_MESSAGES' permission.

Fires mulitply MessageDelete event.

Warning
This method will not delete messages older than 2 weeks, and will fail if any message provided is older than that.
Exceptions
RESTErroron API error (missing permission, invalid ID, older than 2 weeks).
boost::system::system_erroron connection problem (rare).

Send a message with a file to a text channel (or DM).

Warning
Regular accounts and bots have a limit of file size (8 MB), user accounts with Discord Nitro have 50 MB limit. If you send a file bigger than allowed, you will get RESTError with 40005 code.
Exceptions
RESTErroron API error (missing permission, invalid ID, too large file).
boost::system::system_erroron connection problem (rare).
Returns
Message object that represents sent message.
