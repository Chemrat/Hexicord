## echo-bot

Simple bot responds to every message it receives with following:
> Message ID: 357647131575713792<br>
> Channel ID: 342792472306384897<br>
> Sender ID: 130749397372764161<br>
>
> message text

| Environment Variable | Usage                              |
| -------------------- | ---------------------------------- |
| `BOT_TOKEN`          | Bot token.                         |
